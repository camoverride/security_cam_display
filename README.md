# Image Display

Display recent images from a folder to the screen.


## Setup

Install the python requirements: `pip install --file requirements.txt`.


## Run

- `export FLASK_APP=front_end`
- `flask run`
- View in Firefox browser: `http://127.0.0.1:5000`
